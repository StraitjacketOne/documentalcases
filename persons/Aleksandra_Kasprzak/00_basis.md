## Das Muster

1. **Absturz** → Alkoholexzess, Vernachlaessigung des Kindes
2. **Intervention** → JA/Gericht wird aktiv
3. **"Hilfe"** → IF oder Therapie wird angeordnet
4. **"Erfolg"** → Behoerden erklaeren Massnahme fuer erfolgreich
5. **Wiederholung** → Naechster Absturz folgt
6. **Vertuschung** → Behoerden schweigen, Vater wird als Problem dargestellt

Dieses Muster wiederholt sich seit 2021 - dokumentiert ueber 4+ Jahre.
